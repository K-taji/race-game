# 04. テスト計画

## 1. 公開条件

必須テストが通るまで公開しません。暫定決定した機能も初期公開の対象です。

## 2. 機能

| ID | 対象 | テスト | 期待結果 |
|---|---|---|---|
| T-001 | 頭数 | 2頭で開始 | 正常に開始 |
| T-002 | 頭数 | 18頭で開始 | 表示崩れなく開始 |
| T-003 | 頭数 | 1、19、空欄、文字、小数 | 拒否または安全に補正 |
| T-004 | 通常 | 100回実行 | 重複なし、範囲内 |
| T-005 | 推し馬 | ON、7番で複数回実行 | 必ず7番が1着。残りは重複なし |
| T-006 | 推し馬 | ON、未選択 | 開始不可。案内表示 |
| T-007 | 推し馬 | 15番指定後、12頭へ変更 | 選択解除 |
| T-008 | 結果 | 完了 | 1〜5着と補助メモを表示 |
| T-009 | リプレイ | 同じ設定で複数回実行 | 新しい結果を生成 |
| T-010 | スキップ | レース中に`結果を見る` | タイマー停止。同じ確定結果を表示 |
| T-011 | 完了イベント | 自然完了、スキップ | `game_complete`が1回だけ送信 |
| T-012 | 履歴 | 完了後に再読込 | 1件表示 |
| T-013 | 履歴 | localStorage利用不可 | ゲームは継続 |
| T-014 | Reduced motion | 端末設定ON | 動きを短縮または抑制 |
| T-015 | 画像エラー | 馬画像を読込失敗にする | 番号と結果は読める |
