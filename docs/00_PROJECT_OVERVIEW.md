# 00. プロジェクト概要

## 1. 背景

WordPressブログのサブディレクトリで公開している番号抽選Webアプリをリニューアルします。訪問者の多くがWebアプリを利用しているため、抽選機能を残しつつ、番号付きの馬が走るレース演出、任意の推し馬指定、カンパ導線、AdSense運用を追加します。

## 2. 対象環境

| 項目 | 内容 | ステータス |
|---|---|---|
| 現行アプリURL | `https://umano-ie.com/tool01/` | 確定 |
| WordPressサイトURL | `https://umano-ie.com` | 確定 |
| 新版の正規URL | `https://umano-ie.com/race-game/` | 確定。物理フォルダに合わせ末尾スラッシュを使用 |
| レンタルサーバー | Xサーバー | 確定 |
| 原本に記載された更新方法 | `Wordpressのfunction.php` | 原本記録として保持。新版では採用しない |
| 新版の配置方式 | `public_html/race-game/`の独立した静的Webアプリ | 確定 |

## 3. 採用するサーバー構成

```text
対象ドメイン/public_html/
├── WordPress本体                # 原則変更しない
├── .htaccess                    # 原則変更しない。影響だけ確認
└── race-game/
    ├── index.html
    ├── assets/
    │   ├── css/
    │   ├── js/
    │   └── images/
    └── config/
        └── app-config.js
```

### 採用理由

1. WordPressのテーマ更新、プラグイン競合、PHP編集ミスの影響を避けやすい。
2. HTML、CSS、JavaScriptだけで動作し、保守範囲が小さい。
3. `race-game/`フォルダ単位でバックアップと切り戻しができる。
4. カンパ、AdSense、GA4をゲーム本体から分離できる。
5. 現行アプリ`/tool01/`を残したまま新版を確認できる。

## 4. プロダクトの位置付け

番号を付けた競走馬が走る、スマートフォン優先のレトロゲーム風シミュレーターです。遊びとしての抽選と、ユーザーが予想を考える際の補助として使える構成にします。

ただし、実際のレース結果を保証する予想サービスではありません。実在馬名、実際の競馬データ、オッズ、投票、馬券購入機能は扱いません。

## 5. 確定事項

| 項目 | 内容 |
|---|---|
| UI方向 | レトロゲーム風 |
| ギャンブル臭の抑制 | 射幸的な表現を避け、遊びやすいレトロUIにする |
| 馬の識別 | 番号のみ |
| 出走頭数 | ユーザーが2〜18頭で指定 |
| レース時間 | 8〜12秒 |
| 推し馬指定 | 任意で1頭指定。指定番号は必ず1着 |
| カンパ | 任意の応援導線。抽選結果や機能には影響させない |
| カンパサービス | 後から差し替えやすい設計にする |
| AdSense | 継続。後から設定できる設計にする |
| 広告位置 | 初期画面の下 |
| GA4 | 現在導入済み。静的アプリ側でも設定確認する |
| 馬と背景 | AI生成素材 |
| 公開方法 | 現行アプリを残し、新URLで確認後に誘導 |
| 切り戻し | 現行版を維持し、新版フォルダを差し替え可能にする |

## 6. 暫定決定事項

カンパサービス以外の保留項目は、MVP実装の暫定仕様として採用します。

| 項目 | 暫定決定 |
|---|---|
| アプリ名 | `競馬ランダム抽選`。副題: `レトロ競走シミュレーター` |
| 初期頭数 | 12頭 |
| クイック選択 | 5頭、8頭、12頭、18頭 |
| 結果表示 | 1〜5着と`◎○▲△×`の補助表示 |
| 前回結果 | `localStorage`へ1件保存 |
| 馬の表示 | 簡単なアニメーション |
| 順位変動 | あり。中間演出として使用 |
| 実況 | 短文を表示 |
| スキップ | `結果を見る`ボタン |
| 効果音 | 初期版では使用しない |
| カンパ表示位置 | 結果画面の操作ボタン下 |
| 支援者特典 | 初期版では設けない |

## 7. MVP対象外

- 実在馬名、実在馬写真、実際の競馬データ連携
- オッズ、投票、馬券購入サイトへの誘導
- 的中率や勝利確率の向上を示す表現
- 会員登録、データベース、独自決済フォーム
- ランキング、対戦、3Dグラフィック
- 既存ゲーム、競馬団体、公式サービスの模倣
